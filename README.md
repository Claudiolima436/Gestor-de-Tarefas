# Gestor-de-Tarefas
Aplicação Full Stack de Gestão de Tarefas (To-Do List) desenvolvida para demonstrar o domínio do ciclo CRUD. O projeto integra Front-end (HTML/CSS/JS), Back-end (Node/Express) e Banco de Dados (SQLite/MongoDB), comprovando a habilidade de conectar todas as camadas de uma aplicação web."
